# Apache Libcloud (apache-libcloud)
Apache Libcloud is a Python library for interacting with many of the popular cloud service providers using a unified API. It supports over 30 providers for compute, storage, DNS, load balancers, and container services.

## APIs

### Apache Libcloud
Libcloud provides a unified Python API for compute (VM management), object storage, block storage, DNS, load balancer, container, and backup operations across 30+ cloud providers including AWS, Azure, GCP, and OpenStack.
